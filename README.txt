* Some WooCommerce versions may require WooCommerce webhooks to be manually set to Legacy API in WooCommerce, settings > advanced

Major features of PrintAura-Woocommerce include:

* Retrieve products from Printaura.com 
* Place fullfillment requests whenever a PrintAura product is sold through your WooCommerce Shop.

== Installation ==

With WooCommerce installed:

Upload the PrintAura plugin to your blog and activate it.

Some WooCommerce versions may require WooCommerce webhooks to be manually set to Legacy API in WooCommerce, settings > advanced

1, 2, 3: You're done!

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of PrintAura-WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “PrintAura-WooCommerce” and click Search Plugins. Once you’ve found our plugin you can view details about it such as release, rating and description and install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our eCommerce plugin and uploading it to your webserver via your FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
